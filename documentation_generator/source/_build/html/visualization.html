visualization module¶

class visualization.VisualizationModule(image_size=(512, 288))¶

Bases: object

visualize(image, lane_mask, trajectory_coeffs=None, lane_lines=None, y_vals_small=None, show=False, save_path=None)¶

Visualizes the lane detection and planned trajectory on the input image.

Parameters:

  • image – Input image as a NumPy array (BGR format).

  • lane_mask – Lane mask as a NumPy array (same size as image).

  • trajectory_coeffs – Polynomial coefficients of the trajectory.

  • lane_lines – List of polynomial coefficients for lane lines.

  • y_vals_small – Array of y-values in resized image coordinates.

  • show – If True, displays the image in a window.

  • save_path – If provided, saves the visualized image to the specified path.

Returns:

Image with visualizations overlaid.
