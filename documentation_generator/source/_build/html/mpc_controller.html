solve(state, coeffs)¶

Solves the MPC optimization problem.

Parameters:

  • state – Current state [x, y, psi, v, cte, epsi]

  • coeffs – Quadratic polynomial coefficients in vehicle coords in descending order [b2, b1, b0] representing y_v(x_v) = b2*x_v² + b1*x_v + b0.

Returns:

Optimal steering angle (delta) and throttle (a).
